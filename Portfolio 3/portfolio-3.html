lets load all the data:

files <- list.files(path = "data",
                    pattern = ".csv",
                    full.names = T)  

data <- lapply(files, read_csv) %>% 
  plyr::rbind.fill()  

data$Gender = as.factor(data$Gender)
data$Condition = as.factor(data$Condition)

database = read.csv("MRC_database.csv")

data$Word = toupper(data$Word)

# rename the capitalized column to "word"
colnames(data)[colnames(data) == "Word"] = "word"

merged = merge(data, database, by = "word")

Now we have the data and merged it with the MRC_database. First lets look for outliers in the reaction time data since we will be making alot of correlational analysis on that continous varible

As always outliers are defined as -3>z>3

merged$Reaction_timez = (merged$Reaction_time-mean(merged$Reaction_time))/(sd(merged$Reaction_time))

#removing z scores above 3, since there is no z-scores below -3. See plot below

merged1 = filter(merged, merged$Reaction_timez < 3)

ggplot(merged, aes(merged$Reaction_timez))+geom_histogram(binwidth = 1)+labs(y = "frequency", x = "z-scores for reaction time")

Putting the z-score back to normal values.

merged1$Reaction_time2 = merged1$Reaction_timez*sd(merged$Reaction_time)+mean(merged$Reaction_time)

ggplot(merged1, aes(merged1$Reaction_time2)) + geom_histogram(aes(y=..density..), colour = "black", fill = "white", binwidth = 0.1) + labs(x ="Reactiontime", y = "Density") +

Part 1

We now have to make a correlational analysis between Reactiontime and word number, word frequency in the MRC database and number of letters in a word.

Since all the varibles besides reaction time are ordinal, we wont be able to use pearsons correlational test because it assumes that both variables are continuous. Therefore we’ll be using either kendalls tau or spearsmans rho. One would choose kendalls tau over spearmans rho because it is more robust, and is better when there are alot of tied ranks, which we have in the ordinal data.

cor.test(merged1$Reaction_time2, merged1$nlet, method = "kendall")
## 
##  Kendall's rank correlation tau
## 
## data:  merged1$Reaction_time2 and merged1$nlet
## z = 2.2828, p-value = 0.02244
## alternative hypothesis: true tau is not equal to 0
## sample estimates:
##        tau 
## 0.03291127

Visualizing the correlation:

ggplot(merged1, aes(merged1$nlet, merged1$Reaction_time2))+geom_point()+geom_smooth(method = "lm", se = FALSE)+labs(x = "Number of letters", y = "Reaction time in seconds")

Reactiontime was positivly correlated with number of letters in a word with a trivial effect size tau(2698) = 0.033 p < 0.05

Part 2

Since we have a independent design we’ll be using the independent t-test. Which means that in principle we won’t be needing to care about the assumption of homogenity of variance because R uses welchs t-test, which doesn’t assume homogenity of variance, however we’ll be looking at the homogenity of variance just for pratice.

First singling out word number 100(cow/ox) which is the salient word and then, the word after the salient word (word 101 (and))

merged2 = filter(merged1, merged1$X1 == 100)
merged3 = filter(merged1, merged1$X1 == 101)

Before moving on we have to check wether the reactiontimes in the new dataframes are normally distrubuted. First we’ll check for reactiontime for word 100 (cow/ox). I found that the transformation 1/reaction time, gave the most normal data.

con2 = filter(merged2, Condition == 2)

Firstly we see that reaction time in condition 1 for word 100 (ox) is kind of normally distrubuted shaprio-wilks test p > 0.05. kurt.2SE and skew.2SE are also above -1 and below 1. Furthere more the qq-plot looks okay.

We can now see that reaction time in condition 2 for word 100 (cow) is not really normally distrubuted, its significantly different for a perfect normal distrubution (shaprio-wilks test p < 0.05), it is however close to being non significant p = 0.045. It is inside the range of 1 and -1 for kurt.2SE and skew.2SE. However from the qq-plot i would say that the data is not normal, which means we’ll have to use a more robust t-test (yuen)

Since the assumption of normality is violated we won’t be needing to check wether the assumption of homogenity of variance is violated, because the robust t-test (yuen) does not assume that.

T-test for the salient word.
